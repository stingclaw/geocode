GeoCode

A simple open source text editor for OSX﻿

Download this project as a .zip file Download this project as a tar.gz file

GeoCode is a free, open-source plain text editor for OSX, created mostly as a way for me to learn Cocoa programming. It centers pretty highly on keyboard shortcuts, and is currently still in public alpha.
